## 🔑 주요 이슈

### 1. Google Cloud Next 2026 개막 — Gemini Enterprise Agent Platform 공개
- **Google Cloud Next 2026** 라스베이거스 Mandalay Bay 컨벤션 센터에서 4월 22~24일 개최
- 순다르 피차이 CEO 기조연설에서 **Gemini Enterprise Agent Platform** 정식 출시 발표
  - Vertex AI를 대체하는 엔터프라이즈 에이전트 개발 허브
  - **Agent Studio**: 일반 사용자 대상 로코드 시각적 에이전트 설계 도구
  - **Agent Development Kit (ADK)**: 개발자용 SDK
  - **Agent Memory Bank**: 대화에서 장기 기억을 동적으로 생성·관리
  - **Model Armor**: 프롬프트 인젝션·툴 포이즈닝 방어 보안 레이어
- Google의 첫 번째 파티 모델이 분당 **160억 토큰** API 처리 (전 분기 대비 100억 → 60% 성장)
- Gemini Enterprise 유료 MAU 전 분기 대비 **40% 성장** 기록
- **7억 5천만 달러(약 1조원) 파트너 펀드** 조성, 12만 파트너 생태계 지원
- Adobe, Atlassian, Salesforce, Workday 등 주요 SaaS 에이전트 통합 발표
- NVIDIA와 협력해 **물리적 AI 팩토리(Physical AI Factories)** 공동 구축 발표

### 2. Google 내부 코드의 75%가 AI 생성
- 순다르 피차이 CEO 발표: 구글 신규 코드의 **75%를 AI가 생성** (2025년 가을 50%에서 급증)
- AI 에이전트들이 협력해 **코드 마이그레이션을 6배 빠르게** 완료
- 개발자 역할이 코드 작성 → **AI 결과물 검토·지휘**로 전환 가속

### 3. Anthropic — $30B 연간 실행 매출 돌파 & TPU 5GW 계약
- Anthropic의 연간 실행 매출(ARR)이 **$30B 돌파** (2025년말 $9B에서 폭발적 성장)
- Google·Broadcom과 차세대 **TPU 5GW 공급 계약** 체결 (2027년부터 공급)
- Anthropic이 2031년까지 Google에 **커스텀 TPU 설계·공급** 장기 계약
- Broadcom, 2027년말까지 커스텀 AI 칩 사업에서 **연 $1,000억 이상** 매출 전망
- 미공개 모델 **Claude Mythos Preview** 내부 테스트 중:
  - SWE-bench Verified: **93.9%**
  - GPQA Diamond: **94.6%**
  - 주요 OS·브라우저 전반에서 수천 건의 제로데이 취약점 발견

### 4. Alibaba — Qwen3.6-27B, 397B 초대형 모델 능가
- **Qwen3.6-27B** Dense 모델이 에이전트 코딩 벤치마크에서 397B 초대형 모델 초과 달성
  - 크기는 15배 작지만 SkillsBench, SciCode, Terminal-Bench2.0 등에서 우위
- **Qwen3.6-Max-Preview** (4월 20일 출시): 6대 주요 코딩 벤치마크 1위 석권
  - SWE-benchPro, Terminal-Bench2.0, SkillsBench, QwenClawBench, QwenWebBench, SciCode
- 멀티모달(이미지·비디오) + Thinking 모드 지원, **Apache 2.0 라이선스** 완전 무료
- 커뮤니티에서 로컬 최적화(Quantization) 작업 활발 진행 중

### 5. OpenAI — ChatGPT Workspace Agents 출시
- ChatGPT에 **Workspace Agents** 기능 정식 출시 (GPTs의 진화 버전)
- 팀 단위로 **Codex 기반 공유 에이전트** 생성·운영 가능
- 복잡한 멀티스텝 업무를 위한 팀 협업 에이전트 플랫폼

### 6. Meta — AI4AnimationPy 프레임워크 공개
- Python 기반 **3D 캐릭터 애니메이션** 오픈소스 프레임워크 공개
- Unity 같은 무거운 엔진 없이 PyTorch·NumPy만으로 정교한 모션 제어 가능
- **웹 브라우저·스마트폰에서도 경량 구동** 가능
- 파이썬 코드 몇 줄로 가상 캐릭터 조작 환경 구현

### 7. SpaceX — AI 코딩 스타트업 Cursor 인수 추진
- SpaceX가 AI 코딩 도구 **Cursor를 $600억(약 82조원)** 에 인수할 권리 계약
- AI 코딩 도구 시장의 기업가치 폭등을 보여주는 상징적 사례

- **에이전트 AI가 엔터프라이즈 표준으로** — Google Cloud Next 2026의 핵심 메시지: "AI 에이전트는 더 이상 실험적이지 않다, 프로덕션 환경에서 대규모 운영 중"

- **AI 안전·거버넌스 강화** — Google의 Model Armor, Agent Identity (암호화 ID), 이상탐지 등 엔터프라이즈 보안 레이어 본격화
