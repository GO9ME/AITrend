## 🔑 주요 이슈

### 1. ChatGPT 데스크톱 전역 음성 입력 기능 추가

- OpenAI가 ChatGPT 데스크톱 앱에 **전역 음성 입력(Global Voice Input)** 기능 추가
- 단축키 하나로 VS Code, Slack, 웹 브라우저 등 **모든 텍스트 필드에서 즉시 음성 입력** 가능
- 앱 전환 없이 어디서든 AI 음성 입력을 활용할 수 있어 워크플로우 개선 기대
- @choi.openai: "이제 진짜 어디서든 음성으로 작업 가능"

---

### 2. Grok 4.3 출시 — 멀티에이전트 아키텍처로 글로벌 2위

- xAI가 **Grok 4.3**를 공개 — 4개의 전문 에이전트가 협력하는 **멀티에이전트 아키텍처** 도입
- Artificial Analysis 인텔리전스 인덱스 기준 **글로벌 2위(점수 59)** 달성 (1위 GPT-5.5)
- 가격: 입력 $3.00/M tokens
- 코딩, 수학, 추론 작업에서 GPT-5.5와 경쟁하는 수준으로 평가

---

### 3. ClawSweeper — AI 50개 병렬로 하루에 이슈 4,000개 정리

- OpenClaw 개발자가 **ClawSweeper** 프로젝트 공개
- **50개의 Codex AI**를 병렬 운영하여 하루 만에 오픈소스 저장소 내 **4,000개의 불필요한 이슈/PR 정리**
- AI가 코드 작성을 넘어 오픈소스 생태계 **유지보수 업무 자동화**까지 확장 중임을 보여주는 사례
- @choi.openai: "AI 에이전트가 코드베이스 관리 영역에 본격 진입했다"

---

### 4. DeepSeek V4 5월 5일까지 75% 할인

- 전날(4/24) 공개된 **DeepSeek V4-Flash / V4-Pro**가 정식 오픈 기념으로 **5월 5일까지 75% 할인** 프로모션 진행
  - V4-Flash: $0.14/$0.28 → **$0.035/$0.07** per M tokens
  - V4-Pro: $1.74/$3.48 → **$0.435/$0.87** per M tokens
- 이미 낮은 가격에서 추가 할인 → 서방 경쟁 모델 대비 가격 경쟁력 극대화
- 오픈소스(Apache 2.0) + 저가 API = 기업 도입 장벽 최소화

---

### 5. Hermes WebUI — 브라우저 기반 멀티 AI 에이전트 관리 도구

- **Hermes WebUI** 공개: 브라우저에서 여러 AI를 독립적으로 운영하는 에이전트 인터페이스
- 핵심 기능: **다중 프로필** (AI별 독립 운영), **파일 탐색**, **장기 기억 관리** 통합
- **macOS 네이티브 앱**도 동시 출시
- 에이전트 오케스트레이션 도구 시장이 성숙 단계로 진입하고 있음을 보여줌

---

### 6. Federal Reserve 연구 — ChatGPT 이후 프로그래머 채용 성장률 절반으로 감소

- 미국 Federal Reserve 연구: **ChatGPT 출시 이후 프로그래머 채용 성장률이 약 절반 수준**으로 감소

- Meta, Microsoft도 같은 시기 대규모 인력 감축 발표 → AI와의 상관관계 논의 확산
- @choi.openai: "체감이 아닌 데이터로 증명되기 시작했다"
