## 🔑 주요 이슈

### 1. 🚨 속보 — DeepSeek V4 프리뷰 공개: 오픈소스 1M 컨텍스트 MoE

- DeepSeek이 **V4-Flash / V4-Pro** 두 모델로 구성된 V4 프리뷰를 오픈소스 공개
- **V4-Pro**: 총 1.6조 파라미터 / 활성 49B, **V4-Flash**: 총 284B / 활성 13B
- 핵심 기술: **CSA(Compressed Sparse Attention) + HCA(Heavily Compressed Attention)** 하이브리드 어텐션
- **1M 토큰 컨텍스트**를 기본값으로 설계 (후속 추가 기능이 아닌 아키텍처 기본 설계)
- Thinking / Non-Thinking 듀얼 모드, OpenAI·Anthropic API 포맷 동시 지원
- 벤치마크: Putnam-2025 수학 시험 **120/120 만점**, 에이전트 작업에서 "Claude Sonnet 4.5에 근접"
- 라이선스: **Apache 2.0** (상업적 이용 무료), HuggingFace 가중치 공개
- 가격: Flash $0.14/$0.28, Pro $1.74/$3.48 (입력/출력 per M tokens)

---

### 2. 텐센트 Hy3 Preview 출시: 295B MoE, 즉각 10개 제품 통합

- 텐센트가 **Hunyuan Hy3 Preview** 공개 — 야오순위(Yao Shunyu) 합류 후 첫 플래그십 모델
- **구조**: 295B 파라미터 MoE (활성 21B + MTP 3.8B), 256K 컨텍스트
- **성능**: FrontierScience-Olympiad, IMO 수학 벤치마크에서 상위권, 칭화대 수학 박사 자격시험 통과
- **제품 통합 즉시 완료**: Yuanbao(위안바오), WeChat, QQ, CodeBuddy, WorkBuddy, ima, 텐센트 문서 등 10개 이상
  - CodeBuddy 첫 토큰 지연 **54% 감소**, 작업 완료 시간 **47% 단축**, 성공률 **99.99%+**
- **오픈소스**: GitHub/HuggingFace 가중치 공개, OpenRouter API 등록 완료
- 가격: 입력 1.2 위안/M tokens, 출력 약 $0.59/M tokens

### 3. 🚨 빅딜 — Google, Anthropic에 최대 400억 달러 투자 발표

- Google(Alphabet)이 Anthropic에 **최대 400억 달러 규모** 투자 계획 공식 발표
  - **즉시 투자**: 100억 달러 (기업가치 $3,500억 기준)
  - **조건부 추가**: 성능 목표 달성 시 300억 달러 추가
- **컴퓨팅 지원**: Google Cloud가 5년간 **5기가와트(GW)** 컴퓨팅 파워 제공, 추가 수 GW 옵션 포함
- Amazon도 최대 200억 달러 투자 계획 공개 — Anthropic은 빅테크 양쪽에서 동시에 대규모 지원 확보
- **의미**: AI 인프라 전쟁이 '컴퓨팅 계약' 단계로 진화, Claude 모델 훈련 비용 문제 해소 기대

### 5. GitHub 정책 변경 — 비공개 저장소 데이터를 AI 훈련에 활용

- GitHub이 **비공개(private) 저장소 데이터를 AI 모델 훈련에 활용**하는 정책 발표
- 기본값: **옵트아웃(opt-out) 방식** — 거부하지 않으면 자동 동의 처리
- Hacker News에서 폭발적 반응 (677 포인트, 300개 이상 댓글), 개발자 커뮤니티 강한 반발
- **대응법**: GitHub 계정 설정 → Privacy → AI Training 항목에서 비활성화 필요
