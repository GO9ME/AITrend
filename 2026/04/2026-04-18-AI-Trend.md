### 1. Claude Design 시스템 프롬프트 공개 — 내부 설계 철학 유출
- Anthropic의 **Claude Design** 시스템 프롬프트가 GitHub을 통해 외부 공개됨
- "전체적인 디자인 맥락을 깊이 파악하는 데 엄청난 공을 들이고" 있으며, 다양한 대안 탐색을 유도하도록 설계
- 자연스러운 우수 결과는 **"철저하게 기획된 통제와 규칙에서 나온다"** 는 점 강조
- 출처: [GitHub - elder-plinius/CL4R1T4S](https://github.com/elder-plinius/CL4R1T4S)
- choi.openai 코멘트: *"AI 결과물의 품질은 시스템 프롬프트 설계 수준에 달려 있다"*

### 2. Anthropic, Claude Code 글로벌 해커톤 개최
- **Claude Opus 4.7** 기반의 Claude Code 글로벌 해커톤 공식 발표
- 총 **$100,000 규모의 API 크레딧** 제공
- 자동화 스킬 및 에이전트 개발에 초점
- 기존 Claude Opus 4.7 성과: SWE-bench Verified **87.6%**, GPQA **94.2%** 달성
- 컨텍스트 윈도우 **100만 토큰** 지원

---

### 3. VoiceBox — 로컬 실행 오픈소스 TTS 공개
- **23개 언어**, **5개 TTS 엔진** 지원하는 멀티 보이스 합성 도구
- Mac M1 및 일반 PC에서 **로컬 구동** 가능 (클라우드 불필요)
- 수 초 분량의 음성 샘플로 목소리 복제 가능
- 타임라인 편집 기능으로 다중 음성 통합 지원
- 출처: [GitHub - jamiepine/voicebox](https://github.com/jamiepine/voicebox)

---

### 4. Video-Use — AI 기반 영상 편집 자동화 오픈소스
- **"폴더에 원본 던져놓고 말로 지시"** 하면 자동으로 편집 완료
- 공백 제거, 자막 추가, 색보정 자동화 지원
- 영상 프레임 대신 **음성 텍스트 기반 분석**으로 경량 처리
- Claude Code 연동 가능 구조
- 출처: [GitHub - browser-use/video-use](https://github.com/browser-use/video-use)
